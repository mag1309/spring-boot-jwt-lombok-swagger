# spring-boot-jwt-lombok-swagger
This project uses following features
- Spring Security
- Jwt Tokens
- Lombok library
- Spring Validation
- Exception handling
- H2 database
- JPA
- Spring Testing
- Open API
  - Access the API documention at http://localhost:8080/swagger-ui/index.html  
- Actuator 
  - Access the Admin console at http://localhost:8080/actuator
